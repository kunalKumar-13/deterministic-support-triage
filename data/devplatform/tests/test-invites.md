# Sending and managing test invites

A test invite is the link a candidate uses to start an assessment.

## Sending an invite

1. From the workspace dashboard, open the **Tests** tab.
2. Choose the test you want to send.
3. Click **Invite candidate** and enter the candidate's email.
4. The candidate receives an email from `tests@devplatform.com` with a
   one-time-use invite link.

Invite links are valid for 7 days by default. Workspace admins can change
this in **Settings > Tests > Invite link lifetime**.

## Resending an invite

If a candidate does not receive the email, you can resend the same invite
from the **Tests > Candidates** view. The original link remains valid; a
resend does not generate a new link.

## Common issues

- **Candidate cannot click the link**: Check whether the link has expired.
- **Candidate sees a 404**: This can happen if the invite was revoked.
  Issue a new invite.
- **Email never arrives**: Confirm the email address is correct, then
  check the spam folder. If the issue persists, contact support — do not
  manually share the invite link by other means.
