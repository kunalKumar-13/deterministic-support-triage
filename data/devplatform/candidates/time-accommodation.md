# Adding extra time (time accommodation) for candidates

You can grant extra time to one or more candidates for a specific test.
Time is added as a **percentage** of the base test duration; the
configured percentage must be in multiples of 5 (e.g. 25%, 50%, 75%).

## Steps to add time accommodation

1. Log in to your DevPlatform for Work account.
2. Open the **Tests** tab and select the test.
3. Open the **Candidates** tab.
4. Tick the checkbox next to the candidate(s) needing accommodation.
5. Click **More > Add Time Accommodation**.
6. Enter the accommodation percentage (multiples of 5).
7. Click **Save**. A success confirmation appears.

Accommodation can be configured **before** sending the invite as well —
the candidate will then see the adjusted duration in their invite email.

## What the candidate sees

The candidate sees the adjusted duration on the test start screen, even
though the original invite email may show the base duration. This is
expected — the platform applies the accommodation at start time.

## When to reinvite

If the test has not been started, the existing invite stays valid and
the time accommodation will apply automatically — no reinvite needed. If
the test has already been started or completed, reinvitation rules apply
(see test-invites.md).
